# 商户端 Skill - 本地生活 AI 助理

> 帮助商家 7x24 小时自动接待客户的 AI Skill

## 安装

```bash
skill install @local-life-ai/merchant-connector
```

或者手动安装：将 `merchant-connector/` 目录复制到你的 AI Agent 的 skills 目录。

## 功能

- 📊 **数据看板** - 实时查看今日营业数据
- 📋 **排队管理** - 查看当前排队情况
- 📅 **预约管理** - 查看预约到店情况
- 🪑 **桌台状态** - 查看桌台占用情况
- 🔧 **位置校准** - 校准店铺位置，提高推荐精度

## 安装流程

安装时会引导设置：
1. 输入店铺名称
2. 确认店铺地址（自动通过 IP 定位）
3. 校准精确位置（打开校准页面）

## API 配置

安装后，Skill 会自动注册到本地服务注册平台：

```
http://localhost:3000
```

## 商户命令

安装后，商家可以用自然语言：

- **"看看今天的营业情况"** → 数据看板
- **"现在排队多少人"** → 排队列表
- **"今天有几个人预约"** → 预约列表
- **"桌台都坐满了吗"** → 桌台状态
- **"位置不准，帮我校准"** → 打开校准页面

## 数据看板

| 指标 | 说明 |
|------|------|
| 排队中 | 当前等待人数 |
| 今日取号 | 今日总取号数 |
| 今日预约 | 今日总预约数 |
| 待确认 | 需要处理的预约 |

## 文件结构

```
merchant-connector/
├── SKILL.md      # Skill 定义（安装时读取）
├── install.sh    # 安装脚本
├── llm           # 商户命令工具
└── skill.json    # Skill 元数据
```

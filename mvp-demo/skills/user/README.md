# 用户端 Skill - 本地生活 AI 助理

> 帮助用户发现附近好店、排队、预约的 AI Skill

## 安装

```bash
skill install @local-life-ai/user-connector
```

或者手动安装：将 `user-connector/` 目录复制到你的 AI Agent 的 skills 目录。

## 功能

- 🔍 **搜索服务** - 按关键词搜索附近商家
- 📋 **查看排队** - 查看当前排队等待情况
- 🎫 **取号** - 线上取号，到店就餐
- 📅 **预约** - 预约座位，到点用餐

## API 配置

安装后，Skill 会自动连接本地服务注册平台：

```
http://localhost:3000
```

### 健康偏好

首次使用会询问用户：
- 饮食习惯（低油/低糖/素食等）
- 过敏原（坚果/海鲜/乳糖等）
- 健康目标（减肥/增肌/控血糖等）

## 常用 service_id

| 店铺 | service_id | 特色 |
|------|------------|------|
| 金谷园饺子馆 | jin_001 | 低油脂、减肥友好 |
| 宏缘火锅 | hong_001 | 鸳鸯锅、包间 |
| 季多西面馆 | ji_001 | 现拉面、手擀面 |
| 兴华家常菜 | xin_001 | 家常菜、经济实惠 |

## 对话示例

**用户**：帮我找附近的饺子馆

**AI**：帮你搜索一下...

✅ **金谷园饺子馆** - 距离 1.4km
   - 🌟 低油脂，适合减肥人士
   - 📍 海淀区学院路甲38号

**用户**：帮我在金谷园排个号，2位

**AI**：✅ **取号成功！**
- 排队号：**A32**
- 预计等待：**7 分钟**

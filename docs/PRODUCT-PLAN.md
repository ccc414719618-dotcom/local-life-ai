# 本地生活 AI Skill 平台 - 产品规划

> 版本：v0.1 | 日期：2026-04-19 | 作者：基于与用户讨论整理

---

## 1. 产品愿景

**"让每个商家都能用 AI 服务客户，让每个用户都能用 AI 管理本地生活"**

类似 App Store，但对象是 AI Agent。用户对自己的 AI Agent 说一句话，就能完成排队、预约、查询等操作，而不需要打开任何 App。

---

## 2. 核心定位

### 2.1 目标用户

| 用户类型 | 角色 | 需求 |
|---------|------|------|
| **商家** | 餐饮、美发、健身房等本地生活服务商 | 零技术门槛创建 AI 助手，降低客服成本 |
| **用户** | 普通消费者 | 用自然语言管理本地生活服务，不用下 App |
| **AI Agent** | 用户的数字管家 | 发现并调用平台上的 Skill，提供服务 |

### 2.2 解决的问题

| 痛点 | 传统方案 | 我们的方案 |
|------|---------|-----------|
| 商家客服成本高 | 雇人 7x24 | AI 24小时服务 |
| 用户需要下 N 个 App | 美团 + 口碑 + 预约小程序 | 一句话搞定 |
| 平台抽佣高 | 美团抽 20%+ | 平台只收订阅费 |
| 数据不自主 | 平台掌控所有数据 | 数据归商家和用户 |

---

## 3. 功能规划

### 3.1 商家端功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| AI 引导创建 Skill | P0 | 商家描述业务，AI 生成完整 Skill |
| 美团数据迁移 | P1 | 授权后迁移店铺信息、菜单等 |
| 一键部署到本地电脑 | P0 | Docker / Python 脚本部署 |
| 商家后台 | P1 | 管理店铺信息、查看服务统计 |
| 增值服务（高级分析） | P2 | 客流分析、用户画像 |

### 3.2 用户端功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 自然语言发现 | P0 | 「附近哪家饺子馆排队最短」 |
| 一键安装 Skill | P0 | 「帮我安装金谷园饺子馆 Skill」 |
| 预约/排队/查询 | P0 | 核心交易闭环 |
| 用户偏好管理 | P1 | 本地存储，数据归用户 |
| 评价/反馈 | P2 | 帮助商家改进服务 |

### 3.3 平台端功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| Skill 注册表 | P0 | 类似 DNS，黄页性质 |
| MCP 协议兼容 | P0 | 让任何 AI Agent 都能发现 Skill |
| 搜索/发现引擎 | P0 | 按地点、服务类型搜索 |
| 监控/运维 | P1 | 确保平台稳定运行 |

## 4. 行业覆盖规划

### Phase 1（餐饮先行）
- 饺子馆、火锅、烧烤、面馆等
- 核心功能：排队取号、外卖查询、Wi-Fi、营业时间

### Phase 2（生活服务）
- 美发、美容、健身房
- 核心功能：预约、套餐查询

### Phase 3（全行业）
- 诊所、汽修、家政、洗衣等
- 核心功能：预约、取件提醒

## 5. 商业模式

### 5.1 收入来源

| 模式 | 说明 | 优先级 |
|------|------|--------|
| SaaS 订阅 | 商家月费，解锁高级功能 | P0 |
| 交易抽成 | 每笔预约/订单抽 1-3% | P1 |
| 增值服务 | 数据分析、营销工具 | P2 |
| 推广费用 | 商家排名置顶 | P2 |

### 5.2 定价建议

| 套餐 | 价格 | 功能 |
|------|------|------|
| 基础版 | 免费 | 1个 Skill、基础功能 |
| 专业版 | 99元/月 | 无限 Skill、排队功能、数据分析 |
| 企业版 | 299元/月 | 多门店、API 接口、专属客服 |

## 6. 竞争优势

### 6.1 差异化

| 维度 | 美团/口碑 | 我们的平台 |
|------|----------|-----------|
| 交互方式 | App/小程序 | 自然语言，AI Agent 驱动 |
| 数据归属 | 平台所有 | 商家/用户自有 |
| 运营成本 | 高（抽佣 20%+） | 低（订阅制） |
| 技术门槛 | 商家需对接 API | 零门槛，AI 生成 |

### 6.2 护城河

1. **网络效应**：商家越多 → 用户体验越好 → 更多商家入驻
2. **数据锁定**：商家 Skill 一旦建立，迁移成本高
3. **先发优势**：成为赛道第一名

## 9. 风险与应对

| 风险 | 应对措施 |
|------|---------|
| 商家不愿迁移 | 提供数据迁移工具，降低迁移成本 |
| 美团等平台封杀 | 强调数据主权，合规运营 |
| 技术门槛高 | 极致简化 AI 生成流程 |
| 用户习惯难改 | 从极客用户切入，逐步扩散 |

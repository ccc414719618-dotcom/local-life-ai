# 本地生活 AI Skill 平台 - MVP 产品原型

> 版本：v0.2 | 日期：2026-04-20 | 状态：关键决策已确认

---

## 1. 产品定位

**一句话：** "AI时代本地生活的第一个AI通道，让商家的AI和用户的AI自由对接"

**核心价值：**
- 用户：省时间 + 更健康的选择 + 自由选择
- 商家：低成本获客 + AI接待 + 公平竞争

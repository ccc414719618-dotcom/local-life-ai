# 本地生活 AI 平台 - AI-First 网站架构

> 版本：v0.1 | 日期：2026-04-20 | 状态：设计阶段

---

## 1. 核心理念

> **这个网站是给 AI 用的，不是给人用的**
> 
> 先服务 AI，再通过 AI 服务人

**类比：**
- 抖音/小红书 → 人直接看
- 美团 → 人直接用
- **我们的平台** → AI 看 → AI 服务人

---

## 2. 网站定位

| 维度 | 说明 |
|------|------|
| **核心用户** | AI Agent（用户侧 + 商家侧） |
| **次要用户** | 人类（查看/管理/开发） |
| **核心价值** | 让 AI 发现 AI，让 AI 服务人 |
| **数据存储** | 去中心化，平台只做发现层 |

## 6. 核心差异：AI-First vs 传统

| 维度 | 传统网站 | AI-First 网站 |
|------|---------|--------------|
| **主要用户** | 人 | AI Agent |
| **HTML** | 核心资产 | 辅助展示 |
| **JSON/MCP** | 可选 | 核心优先 |
| **搜索** | 关键词匹配 | 语义+结构化 |
| **数据存储** | 中心化 | 去中心化 |
| **交易** | 平台完成 | AI-Agent 完成 |

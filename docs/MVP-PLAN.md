# 本地生活 AI Skill 平台 - MVP 落地执行方案

> 版本：v0.1 | 日期：2026-04-19 | 作者：基于与用户讨论整理

---

## 1. MVP 目标

**4周内，完成一个可运行的「餐饮行业 AI Skill 平台」Demo**

- ✅ 商家能零门槛创建自己的餐饮 Skill
- ✅ Skill 能部署到商家电脑
- ✅ 用户能找到并安装 Skill
- ✅ 用户能通过 AI 完成排队、查询等操作
- ✅ 3-5家真实商家内测

---

## 2. MVP 范围

### 2.1 做（In Scope）

| 模块 | 功能 | 优先级 |
|------|------|---------|
| **Skill 生成器** | AI 引导创建餐饮 Skill | P0 |
| **MCP Server 模板** | 标准化的商家服务端 | P0 |
| **Skill 注册表** | 简单网页展示 + JSON API | P0 |
| **Tailscale 部署** | 商家电脑一键暴露 MCP | P0 |
| **排队功能** | 接入 meituan-queue | P1 |
| **商家后台** | 修改店铺信息 | P1 |

### 2.2 不做（Out of Scope）

| 功能 | 原因 |
|------|------|
| 美团数据迁移 | 太复杂，MVP 后做 |
| 支付闭环 | 太复杂，MVP 后做 |
| 评价系统 | MVP 后做 |
| 多行业支持 | 先专注餐饮 |
| 用户App | 用网页展示即可 |

## 6. 商家入驻流程（傻瓜版）

```
Step 1: 打开商家后台
        ↓
Step 2: 用手机号登录
        ↓
Step 3: AI 问你问题
        「你开的是什么店？」
        输入：饺子馆
        ↓
Step 4: AI 继续问
        「店铺叫什么名字？」
        「地址在哪？」
        「营业时间？」
        ↓
Step 5: AI 生成 Skill
        「已为你生成金谷园饺子馆 Skill」
        ↓
Step 6: 一键部署
        点击「部署到我的电脑」
        ↓
Step 7: 下载并运行
        商家下载程序 → 运行 → 自动注册到平台
        ↓
        ✅ 完成！
```

## 7. 用户使用流程

```
场景：用户想找附近能排队的饺子馆

Step 1: 对 AI 说
        「帮我找附近能排队的饺子馆」
        ↓
Step 2: AI 读取 Skill Registry
        发现「金谷园饺子馆」支持排队
        ↓
Step 3: AI 安装 Skill
        「已安装金谷园饺子馆 Skill」
        ↓
Step 4: 用户查询
        「现在排队情况怎么样」
        ↓
Step 5: AI 调用商家 MCP Server
        获取实时排队数据
        ↓
Step 6: AI 返回结果
        「金谷园当前无需等待，有空桌」
        ↓
Step 7: 用户继续操作
        「帮我在那排个2人桌」
        ↓
Step 8: AI 完成取号
        「已取号，排队号 A12，预计等待 0 分钟」
```

## 9. 风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 商家电脑无法暴露端口 | 中 | 中 | Tailscale 中继模式 |
| Skill 生成质量差 | 低 | 高 | 人工审核 + 模板优化 |
| 商家不愿意部署 | 中 | 高 | 提供上门部署服务 |
| 技术难度超预期 | 低 | 中 | 砍功能，先跑通再说 |
